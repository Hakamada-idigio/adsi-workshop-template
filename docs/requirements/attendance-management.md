# 勤怠管理アプリ — 要求仕様書

## 概要

社員が日々の出退勤・休憩を打刻し、管理者が勤怠を承認・集計できる Web アプリケーション。
フレックスタイム制に対応し、100人規模の組織で PC・スマホから利用する。

## ユーザーと権限

| ロール | できること |
|--------|-----------|
| 一般社員 | 出退勤・休憩の打刻、自分の勤怠確認・修正、有給/残業/休日出勤の申請 |
| 管理者 | 上記に加え、部下の勤怠承認、打刻修正の承認、月次レポート閲覧、CSV エクスポート、残業アラート確認 |

- 認証: ID（社員番号）＋ パスワード
- 社員番号は既存番号を使用（自動採番ではない）

## ユーザーストーリー

### 打刻

- **US-001**: 社員として、出勤ボタンを押して出勤時刻を記録したい
- **US-002**: 社員として、退勤ボタンを押して退勤時刻を記録したい
- **US-003**: 社員として、休憩開始・終了ボタンを押して休憩時間を記録したい
- **US-004**: 社員として、承認前の打刻を自分で修正したい
- **US-005**: 社員として、承認後の打刻修正を申請したい（管理者承認が必要）

### 勤怠確認

- **US-006**: 社員として、自分の日別・月別の勤務実績を一覧で確認したい
- **US-007**: 社員として、フレックスのコアタイム・総労働時間の過不足を把握したい

### 申請・承認

- **US-008**: 社員として、有給休暇を申請したい
- **US-009**: 社員として、残業を申請したい
- **US-010**: 社員として、休日出勤を申請したい
- **US-011**: 管理者として、部下の有給/残業/休日出勤の申請を承認・却下したい
- **US-012**: 管理者として、部下の打刻修正申請を承認・却下したい

### レポート・エクスポート

- **US-013**: 管理者として、部下の月次勤怠集計レポートを閲覧したい
- **US-014**: 管理者として、勤怠データを CSV エクスポートしたい

### アラート

- **US-015**: 管理者として、36協定の残業上限に近づいている社員にアラートを出したい

### 認証

- **US-016**: ユーザーとして、社員番号とパスワードでログインしたい
- **US-017**: ユーザーとして、パスワードを変更したい

## ビジネスルール

| # | ルール |
|---|--------|
| BR-001 | 勤務形態はフレックスタイム制（コアタイム 10:00〜15:00） |
| BR-002 | 勤務時間は1分単位で記録（丸めなし） |
| BR-003 | 月の所定労働時間 = 7.25時間 × 当月の営業日数 |
| BR-004 | 承認前の打刻（出退勤・休憩とも）は本人がいつでも修正可能 |
| BR-005 | 承認後の打刻修正は本人申請 → 管理者承認が必要 |
| BR-006 | 勤怠の提出は社員が月末にまとめて「提出」ボタンを押す（DRAFT → SUBMITTED） |
| BR-007 | 勤怠データは労基法に基づき3年間保存する |
| BR-008 | 36協定の残業上限（月45時間/年360時間）を超過しそうな場合にアラートを出す |
| BR-009 | 一覧 API はページネーション対応とする |

## 非機能要件

- **対応デバイス**: PC ブラウザ + スマートフォンブラウザ（レスポンシブ）
- **想定ユーザー数**: 100人規模
- **外部連携**: なし（スタンドアロン）
- **データ保存期間**: 3年間（労基法準拠）

## MVP 段階分け（提案）

### Phase 1: 基本打刻（MVP）

最小限の動くものを最短で作る。

- ログイン（社員番号 + パスワード、JWT 8時間有効・リフレッシュなし）
- 出勤・退勤・休憩開始・休憩終了の打刻
- 自分の日別勤怠一覧表示（ページネーション対応）
- 承認前の打刻修正（出退勤・休憩とも、本人が同じ編集画面から修正）
- 月末の勤怠提出（DRAFT → SUBMITTED）

### Phase 2: 承認フロー

管理者による運用を可能にする。

- 管理者ロールの追加
- 月次勤怠の承認・差し戻し
- 承認後の打刻修正申請・承認
- 部下の勤怠一覧閲覧

### Phase 3: 申請機能

各種申請ワークフローを追加する。

- 有給休暇の申請・承認
- 残業申請・承認
- 休日出勤の申請・承認

### Phase 4: レポート・アラート

集計・可視化機能を追加する。

- 月次勤怠集計レポート
- CSV エクスポート
- 36協定残業アラート
