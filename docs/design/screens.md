# 画面設計

## 画面一覧

| # | 画面 | ロール | Phase | 主な機能 |
|---|------|--------|-------|---------|
| S-01 | ログイン | 全員 | 1 | 社員番号 + パスワード入力 |
| S-02 | ダッシュボード（打刻） | 全員 | 1 | 出退勤・休憩ボタン、当日の状態表示 |
| S-03 | 日別勤怠詳細 | 全員 | 1 | 打刻時刻・休憩時間の確認・修正 |
| S-04 | 月別勤怠一覧 | 全員 | 1 | カレンダー形式で月の勤怠一覧 |
| S-05 | パスワード変更 | 全員 | 1 | 現在のパスワード + 新パスワード |
| S-06 | 承認一覧（管理者） | 管理者 | 2 | 部下の勤怠・申請の承認待ち一覧 |
| S-07 | 部下の勤怠一覧 | 管理者 | 2 | 部下ごとの月別勤怠閲覧 |
| S-08 | 申請作成 | 全員 | 3 | 有給・残業・休日出勤の申請フォーム |
| S-09 | 申請一覧 | 全員 | 3 | 自分の申請状況確認 |
| S-10 | 月次レポート | 管理者 | 4 | 集計テーブル + CSV ダウンロード |
| S-11 | 残業アラート | 管理者 | 4 | 36協定超過リスクの社員一覧 |

## レスポンシブ対応方針

- **PC**: サイドバー + メインコンテンツのレイアウト
- **スマホ**: ハンバーガーメニュー + シングルカラム
- 打刻ボタンはスマホでも大きく押しやすいサイズにする
- テーブルはスマホではカード形式に切り替え

## コンポーネント一覧

| コンポーネント | 用途 |
|--------------|------|
| ClockButton | 出退勤・休憩の打刻ボタン |
| AttendanceTable | 月別勤怠テーブル |
| AttendanceCard | スマホ用の日別勤怠カード |
| ApprovalList | 承認待ち一覧 |
| LeaveRequestForm | 申請フォーム |
| MonthPicker | 年月選択 |
| StatusBadge | 状態表示バッジ（DRAFT/APPROVED 等） |
| WorkTimeSummary | 労働時間サマリー |
| OvertimeAlertBanner | 残業アラート表示 |
